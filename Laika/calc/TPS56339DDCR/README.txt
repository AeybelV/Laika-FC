This folder contains the result from utilizing Texas Instruments WEBENCH Power Designer

Requirements specified:

Input: DC 5V - 12V
Output: 3.3V at 2A
Temp: 30 C

Recomended Schematic and Component Values can be found in TIWBDesign.pdf

Recomended Layout can be found in TOP.svg and BOT.svg

TI WEBENCH is a nice utility for designing power circuits using TI Components. If you wish
to tweak the requirements for your build, feel free to utilize WEBENCH to provide you component
values for this regulator that meets your demands.
